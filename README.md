# 🚴 Bike Sales Analysis Dashboard

## 📌 Objective

The goal of this project is to analyze customer data and identify the key factors that influence bike purchasing behavior.
This analysis aims to support data-driven decisions for improving sales and targeting the right customer segments.

---

## 📊 Dataset Description

The dataset includes customer-level data such as:

* Demographics (Age, Gender, Marital Status)
* Financial information (Income)
* Lifestyle factors (Number of Cars, Commute Distance)
* Purchase decision (Purchased Bike)

---

## 🧹 Data Cleaning & Preparation

* Converted income from text format to numeric values
* Standardized categorical values using Find & Replace
* Removed duplicate records to ensure data accuracy
* Created age groups using Nested IF formulas
* Created an ordered column for Commute Distance to enable proper sorting

---

## ❓ Business Questions

* Does commute distance impact bike purchasing behavior?
* Which gender is more likely to purchase bikes?
* Which age group has the highest purchase rate?
* How does car ownership affect purchasing decisions?

---

## 📈 Key Insights

### 🚶 Commute Distance

Customers with longer commute distances are less likely to purchase bikes.

### 👨 Gender

Male customers show a higher bike purchase rate compared to female customers.

### 🚗 Car Ownership

Customers owning more than two cars are less likely to purchase bikes.

### 👥 Age Group

Middle-aged customers (30–50) represent the highest purchasing segment.

---

## 💡 Recommendations

* 🎯 Target short-distance commuters with bike promotions
* ⚡ Introduce electric bikes (E-bikes) to attract long-distance commuters
* 👩 Increase marketing efforts toward female customers to improve engagement
* 🚗 Focus on customers with fewer cars as a primary target segment
* 👥 Prioritize middle-aged customers in marketing campaigns

---

## 🛠 Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Data Visualization (Charts)
* Nested IF formulas (for segmentation)
* Data Cleaning techniques (Find & Replace, Remove Duplicates)

---

## 📊 Dashboard

The dashboard provides an interactive view of customer behavior using filters such as:

* Marital Status
* Region
* Number of Children

It helps identify patterns and trends in bike purchasing decisions.

---

## 📌 Conclusion

This project demonstrates how data analysis can uncover valuable insights about customer behavior.
By applying targeted strategies based on these insights, businesses can improve conversion rates and optimize their marketing efforts.
